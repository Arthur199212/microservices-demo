# API Gateway Service

Used as an entry point for clients to communicate with the rest of the application. It's built as RESP API and uses JSON for communication.

## Technologies Used

- [gRPC](https://grpc.io/)
- [fiber](https://github.com/gofiber/fiber)
- [viper](https://github.com/spf13/viper)
- [zerolog](https://github.com/rs/zerolog)
- [go-playground/validator](https://github.com/go-playground/validator)
